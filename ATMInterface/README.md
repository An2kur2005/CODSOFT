🏦 ATM Interface System
A sleek, secure, and user-friendly ATM Simulation that brings the banking experience right to your terminal. Whether you're checking your balance or making a "virtual" fortune, this interface handles it all with ease! 💸

📝 Description
This project is a functional ATM (Automated Teller Machine) interface. It mimics the core logic of a real-life bank machine, allowing users to perform essential financial transactions. It's designed with a focus on clean logic, secure input handling, and a smooth user flow. 💳✨

✨ Features

💰 Balance Inquiry: Check your current funds in real-time.

📥 Cash Deposit: Seamlessly add "funds" to your account balance.

📤 Cash Withdrawal: Take money out (with built-in insufficient funds protection!).

📜 Transaction History: Keep track of your latest deposits and withdrawals.

🚪 Safe Exit: Securely end your session and log out.

🛠 Tech Stack
Built with reliability and simplicity in mind:

Language: Java 

Logic: Object-Oriented Programming (OOP) 🏗️

IDE: eclipse
