🎮 Number Guessing Game
Welcome to the Number Guessing Game! This is a simple, fun, and interactive command-line game where the computer picks a secret number, and you try to figure out what it is. 🕵️‍♂️

📝 Description
Have you ever wanted to test your intuition? This game challenges you to guess a randomly generated number within a specific range. The computer provides real-time feedback—telling you if your guess is "Too High" or "Too Low"—until you hit the bullseye! 🎯

✨ Features
🎲 Randomized Gameplay: A new number is generated every time you play.

💡 Smart Hints: Immediate feedback to help narrow down your search.

🔢 Score Tracking: Keeps track of how many attempts it took to win.

🔄 Play Again: Easily restart a new round without exiting the app.

⚡ Input Validation: Prevents the game from crashing if you accidentally type a letter instead of a number.
