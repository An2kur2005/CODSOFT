📊 Student Grade Calculator
A simple yet powerful Java-based tool designed to automate academic result calculations. This project was developed to help students and educators quickly determine total marks, average percentages, and corresponding letter grades based on user input.

📝 Description
The Student Grade Calculator is a console-based application that takes marks obtained in various subjects (out of 100) as input. It then processes this data to calculate the overall performance and provides a final grade. This project focuses on input validation, arithmetic logic, and clean console output.

✨ Key Features
🔢 Multi-Subject Support: Input marks for any number of subjects.

📈 Automatic Calculation: Instantly calculates Total Marks and Average Percentage.

🎯 Grade Assignment: Automatically assigns grades (e.g., A, B, C) based on the calculated percentage.

🛠️ Error Handling: Basic checks to ensure marks entered are within a valid range (0–100).

🖥️ User-Friendly Interface: Clear, readable prompts and a formatted result summary.

🛠️ Technical Stack & Tools
Language: Java 

Concepts: Loops, Conditional Statements, and Scanner Class.

IDE: Eclipse
