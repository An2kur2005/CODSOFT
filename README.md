🚀 Projects Overview

1️⃣ Number Guessing Game
An interactive game designed to test logic and random number generation.

2️⃣ Student Grade Calculator
A utility tool for academic performance tracking and grade distribution.

Features: Multi-subject input, total marks calculation, average percentage, and automated grade assignment.

Concepts: Input validation, Arrays, and Arithmetic operations.

3️⃣ ATM Interface
A simulation of a real-world Automated Teller Machine system.

Features:Balance inquiries, deposit/withdrawal logic, and transaction history.

Concepts: Encapsulation, Methods, and Object-Oriented Programming (OOP).
